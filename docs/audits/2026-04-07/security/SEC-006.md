# SEC-006: Dependency Vulnerabilities (rollup, esbuild, vite)

- **Severity:** HIGH (development only)
- **Category:** Supply Chain / Dependencies
- **Source:** `pnpm audit` (2026-04-07)

## Findings

| Package | Severity | Version | Advisory | Path |
|---------|----------|---------|----------|------|
| rollup | HIGH | `<4.59.0` | [GHSA-mw96-cpmx-2vgc](https://github.com/advisories/GHSA-mw96-cpmx-2vgc) — Arbitrary file write via path traversal | `stoat-worker > tsup > rollup` |
| esbuild | MODERATE | `≤0.24.2` | [GHSA-67mh-4wv8-2f99](https://github.com/advisories/GHSA-67mh-4wv8-2f99) — Dev server request bypass | `api-docs > vitepress > vite > esbuild` |
| vite | MODERATE | `≤6.4.1` | [GHSA-4w7w-66w2-5vf9](https://github.com/advisories/GHSA-4w7w-66w2-5vf9) — Path traversal in .map handling | `api-docs > vitepress > vite` |

## Production Impact

**NONE** — All 3 vulnerabilities are in development dependencies:

1. **rollup** is a build tool used by `tsup` to bundle the `stoat-worker`. The vulnerability requires crafting malicious build input, which is controlled by the developer.
2. **esbuild** and **vite** are used by VitePress for the `api-docs` development server. The vulnerability affects the dev server only, not production builds.

No production Cloudflare Workers are affected.

## Recommendation

1. **stoat-worker:** Update `tsup` to a version that pulls `rollup ≥4.59.0`
   ```bash
   pnpm --filter xivdyetools-stoat-worker update tsup
   ```

2. **api-docs:** Update `vitepress` to latest for esbuild/vite fixes
   ```bash
   pnpm --filter xivdyetools-api-docs update vitepress
   ```

3. **CI integration:** Add `pnpm audit --production` to CI pipeline (ARCH-003 from 2026-03-18 audit, still open)

## Effort

LOW — Package version bumps only.
